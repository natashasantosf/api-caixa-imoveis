# api-caixa-imoveis
API simples para listar imóveis à venda da Caixa Econômica Federal

Regularly fetch all tables from Leilões da Caixa and serve it as an API.

Data Source: https://venda-imoveis.caixa.gov.br/sistema/download-lista.asp

## Expected result
OUTPUT: JSON data complying with the API2MAP Standard.

## The API2MAP Standard
The API2MAP Standard is a type definition for json data.
This is a standard to easily display data in map websites.

You’re going to invent this definition, so you can use it in a data2map website (like the one you’re doing now).
